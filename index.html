KPC

KPC is an online non-veg food website where user can buy different kinds of chicken dishes and get delivered at the door-step within 24hrs. The website showcase variety of chicken items with different flavours, sizes, price etc. The user can buy any product, mark any product as favourite, get the summary of the order and make the payment. The website also has admin side where admin can add, delete, update and create new prodcuts in the database.

Fashion Fabric

Fashion Fabric is an e-commerce website that provides modern clothing like coats, jackets, pants etc. The websites provides clothing for different age group, gender, style, colors etc. User can search the desired apparels, fine tune the results using filter functionality and sort according to the price or rating of the products. User can buy multiple products, mark them as favourite, see the order details, delete the product from the order and make the payment. The website also has the admin side where stock can be managed by adding, updating, deleteing or creating new products in the database
