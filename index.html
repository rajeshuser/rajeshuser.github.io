Chillout

Tech Stack: HTML | CSS | JS

GitHub Link Site Link

Chillout is an online platform where user can search and book any hotel. User can search multiple hotels across the world at different locations. The hotels can be find by using specific keywords. Hotels list can also be fine tuned using multiple filters based on price, class, deals, brands, etc. Any hotel can booked and view them in the cart.

KPC

Tech Stack: HTML | CSS | JS

GitHub Link Site Link

KPC is an online non-veg food website where user can buy different kinds of chicken dishes and get delivered at the door-step within 24hrs. The website presents variety of chicken items which can be searched by keywords and filtered for fine tuned results. User can purchase any product, mark as favourite, get the order summary and make the payment. The website also has admin side where admin can manage and update the stock.

Fashion Fabric

Tech Stack: HTML | CSS | JS | React

GitHub Link Site Link

Fashion Fabric is an e-commerce website that provides modern clothing like coats, jackets, pants etc. User can search the desired apparels, fine tune the results using filter functionality and sort according to the price or rating of the products. User can buy multiple products, mark them as favourite, see the order details, delete the product from the order and make the payment. The website also has the admin side to manage the stock in the database.
